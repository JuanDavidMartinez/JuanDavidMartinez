Guia 3.0 prototipo inicial

Ejercicio:Crear una lista de elementos HTML y explorar el uso de selectores:

  1. Selectores simples:

    Este selector hace referencia directamente a un tipo de elemento HTML. Es la manera más sencilla para referirse a todos los elementos de un mismo tipo, como ejemplo este parrafo se encuentra de color rojo, con un tamaño de letra en el archivo CSS de 20 px y con fuente Helvetica,por medio de la implemantación de un selector simple.

  2. Selector de clase:

    Se forma con un punto, '.', seguido de un nombre de clase. Un nombre de clase puede ser cualquier valor sin espacios usado dentro de un atributo HTML class. Podemos elegir el nombre que deseemos para la clase. Es conveniente saber que en un mismo documento varios elementos pueden compartir el mismo valor de clase y que un elemento puede tener varios nombres de clases separados por un espacio en blanco.

  3. Selectores ID

    El selector ID está formado por una almohadilla (#), seguida del nombre ID de determinado elemento. Cualquier elemento solo puede tener un único nombre ID fijado con el atributo id. Podemos usar cualquier nombre para el ID. Es la forma más efectiva de selecionar un solo elemento.
